# Ferramenta_de_Controle_de_Investimentos
Uma ferramenta em Microsoft Excel para gestão de investimentos, voltada ao controle financeiro e apoio à tomada de decisão. possibilita a simulação da projeção do patrimônio acumulado, o acompanhamento de dividendos e rendimentos periódicos, além da classificação do perfil do investidor, oferecendo sugestões de alocação compatíveis.
